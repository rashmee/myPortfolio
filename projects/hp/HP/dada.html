Information about the Defence Against the Dark Arts class:

About the Class The student will be taught to defend themselves from the dark arts and may dark creatures using magic. Get ready to face boggarts, and learn about many curses like the cruciatus curse and the avada kedavara. This class is madatory from years one through five.
Professors Due to the position being jinxed there have been various professors and many more to come till one of them remains.

  1. Quirinius Quirrell


  2. Gilderoy Lockhart


  3. Remus Lupin


  4. Barty Crouch Jr. impersonating Alastor Moody


  5. Severus Snape

All of these Professors have been highly qualified... except for Barty Crouch Jr. who's qualifictions are still at questions as he spent a lot of his time at Azkaban. Despite that the student learnt a lot from this class.
Class Details Class Location This class meets in classes 31, 104, 3C. This class usually meets various times in the week. Please refer to schedule on the first day of class.
